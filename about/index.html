This site is built to showcase the Hexo theme Meow, while also providing theme documentation and some example pages for quick reference and usage.
If you enjoy a clean, cute, and mobile-friendly style when building or writing, this theme should suit your needs.

Theme Features (Overview)

  • Rich configuration options, easy to toggle and customize;
  • Responsive layout for both mobile and desktop;
  • Built-in tag plugins for flexible use in posts;
  • Special optimization for serials/fanfiction or other long-form content;
  • Archive page with card view by year/month;
  • Built-in dynamic page and links page;
  • Supports TOC (Table of Contents);
  • Floating toolbar and donation module;
  • One-click light/dark mode switch with customizable primary color;
  • Integrated comment systems;
  • Image lazy loading and lightbox viewing;
  • Site search;
  • Multi-language (i18n) support;
  • Compatible with post encryption plugins;
  • Built-in 404 page;
  • SEO friendly.

Theme Statement (Summary)

  1. When using Meow, please keep the theme copyright information in the site footer. This is both a sign of respect to the author and a way for more users to discover the theme.
  2. Secondary development and modification are allowed, but you must clearly credit the original project (hexo-theme-meow) and author (chanwj), and retain the original copyright and license statements.
  3. For issues or suggestions, please first submit an Issue on GitHub. You may also leave a comment on the site or contact the author through other channels.
  4. If you find the theme useful, consider giving it a Star on GitHub to support the author.

Note: The above summary is based on the publicly available page and is not a verbatim copy. Please always refer to the official site and repository for the latest and most accurate information.
